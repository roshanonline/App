# Echo Memory Backend API

This directory contains the Node.js/Express backend for the Echo Memory game.

## Features

*   User Authentication (Register/Login) using JWT
*   Score Saving
*   Leaderboard Retrieval

## Setup

1.  Install dependencies: `npm install` or `yarn install`
2.  Create a `.env` file in this directory (based on the example).
3.  Add your MongoDB connection string and a strong JWT secret to the `.env` file.
4.  Run in development: `npm run dev` (uses nodemon)
5.  Run in production: `npm start`

## API Endpoints

*   `POST /api/auth/register`: Register a new user.
*   `POST /api/auth/login`: Login an existing user, returns JWT.
*   `POST /api/game/save-score`: Saves a user's score (Requires Authentication).
*   `GET /api/leaderboard`: Gets top scores (Public). Supports `limit`, `filter` (daily, weekly, all-time), and `mode` query parameters.
